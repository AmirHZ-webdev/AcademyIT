آموزش کتابخانه Less

LESS یکی از کتابخانه های CSS است که از قوانین CSS استفاده می‎کند؛

هیچ صفحه وب سایتی نداریم که از css استفاده نکرده باشد چون همه ویژگی های رنگ و فونت و عملیات ریسپانسیو سازی صفحات سایت با css صورت میگیرد بنابراین css یک بخش جدانشدنی از صفحات وب سایت هاست، اما استفاده از css به تنهایی یکم مارو محدود تر میکند؛ به همین دلیل کتابخانه هایی تحت محور css به وجود آمدند مانند less تا به ما کمک کنند مدیریت بهتری بر کدهای css داشته باشیم؛ در این دوره آموزشی همراه شماییم با آموزش کتابخانه Less.

Less چیست؟

Less یک کتابخانه css مبتنی بر جاوااسکریپت است و با استفاده از این کتابخانه میتوینم با بهره مندی از الگوریتم ها و توابع مختلف استایل دهی خود را مدیریت کنیم اما نیاز به یادگیری جاوااسکریپت نیست. ینی شما بدون نیاز به یادگیری زبان برنامه نویسی میتونید از امکاناتی نظیر حسابگر، mixin، توابع و.. استفاده کنید چون که کامپایلر Less بر اساس جاوااسکریپت نوشته شده. شما با دیدن دوره آموزشی less میتوانید کد های css پویا بنویسید، و در پایان پروژه بسایر راحت تر از قبل میتونید کد های css خود را مدیریت کنید

چرا باید از Less استفاده کنیم؟

  1. جلوگیری از تکرار کد های css
  2. منظم بودن کد های استایل دهی
  3. ابزاری برای مدیریت راحت ترcss
  4. صرفه جویی در زمان
  5. استفاده از متغیر
  6. ایجاد توابع
  7. استفاده از توابع رنگی
  8. استفاده ار حلقه ها
  9. استایل دهی تو در تو برای عناصر
